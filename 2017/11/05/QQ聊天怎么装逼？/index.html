安装使用环境

安装教程

安装perl

一般来说，Windows系统是没有安装这个perl的，所以我们需要自己装上去，我选择的是ActivePerl

安装dmake

安装好perl后，去执行perl xxxx.pl
命令时会提示你有些东西没安装上，所以我们需要手动安装dmake。
打开cmd，执行ppm install dmake，安装完之后输入dmake -V【注意是大写的V】，显示如下内容则说明安装成功了

安装cpan

首先去这里找到最新的安装包，名字类似于这样的：App-cpanminus-x.xxxx.tar.gz，下载下来后解压到方便的文件夹，然后用cmd进入这个文件夹，依次执行下列命令
perl Makefile.PL
dmake test
dmake install

安装cpanm包管理工具

执行cpan -i App::cpanminus
等待完成即可

安装Webqq模块

执行cpanm Mojo::Webqq
等待完成即可
